ScatterWidget#

class napari_matplotlib.ScatterWidget(napari_viewer: napari.Viewer, parent: Optional[PyQt5.QtWidgets.QWidget] = None)#

Bases: napari_matplotlib.scatter.ScatterBaseWidget

Scatter data in two similarly shaped layers.

If there are more than 500 data points, a 2D histogram is displayed instead of a scatter plot, to avoid too many scatter points.
